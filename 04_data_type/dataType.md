# Data Type

## Basic types of data type in Go:

-	int: stores whole number (negative non negative both)

-	float: stores decimal number (negative non negative both)

-	bool: stores boolean value (true or false)

-	string: stores sequence of characters (text)


###	int:
there are 2 types of int data type :

-	signed integar: stores negative and non-negative both type of values

-	unsigned integar: stores only non-negative type of values. we use 
uint keyword to assign unsigned integer
	
###	float
there are 2 types of float data type:

-	float32: it is of size 32 bit
-	float64: it is of size 64 bit
